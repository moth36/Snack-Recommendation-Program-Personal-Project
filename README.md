모듈은 따로 설치하셔야 합니다.

Node.js와 Express를 활용한 개발을 했습니다.

2학년 기말 프로젝트로 과자 추천 및 구매 웹 서비스를 개인 개발하였으며,
Node.js와 Express를 이용해 서버를 구축하고,
MySQL2를 사용하여 사용자 정보와 상품 데이터를 관리하였습니다.

해당 프로젝트에서는 과자 목록 조회, 검색 필터, 구매 기능을 구현하였고,
프론트엔드는 EJS 템플릿 엔진을 활용하여 직관적인 UI를 구성하였습니다.
당시 과자 데이터를 CSV로 자동 처리하는 방식도 고민했으나,
시간 제약으로 인해 직접 DB에 데이터를 입력하였고,
추후에는 CSV 자동 입력 기능을 구현해 보고 싶다는 목표를 갖게 되었습니다.

또한 현재는 로컬 서버 기반으로 동작하지만,
향후 Socket.io를 활용하여 실시간 통신 기반 서비스로 확장하고 싶습니다.

이러한 프로젝트 경험을 통해
웹 서비스 설계, 서버와 데이터베이스 연동, 실시간 통신 처리에 대한 이해를 높일 수 있었습니다.
